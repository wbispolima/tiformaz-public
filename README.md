# Projeto FORMAZ — Organização Geral

## Visão geral

Este repositório centraliza a documentação e a organização dos processos da Formaz.
Aqui ficam registradas informações sobre infraestrutura, sistema de tarefas, fluxos internos, padronizações e procedimentos operacionais.

O objetivo é garantir continuidade, facilitar manutenção e permitir que qualquer membro da equipe consiga entender rapidamente como o ambiente funciona.

---

- [Infraestrutura](infraestrutura/README.md)  
  Documentação de servidores, serviços, automação (Terraform/Ansible), Redmine e demais componentes de infraestrutura.

- [Sistema de Tarefas](sistema-tarefas/README.md)  
  Organização do modelo de atendimento de demandas, tipos, categorias, fluxos e regras de uso.

- [Padronizações](padronizacoes/README.md)

- [Procedimentos Operacionais](procedimentos/README.md)  
  Rotinas de manutenção, atualização, backup, migração e checklists.
